## 🎯 PHASE 1: Google Search Console (24-48 hours)

### 1.1 Verify Domain Ownership ✅ (Priority: CRITICAL)
- [ ] Go to [Google Search Console](https://search.google.com/search-console/)
- [ ] Select your property: `officeaddindevelopment.com`
- [ ] Verify status shows "Verified"
- [ ] **If not verified**: Go to Settings → Ownership Verification → Add verification method

### 1.2 Submit Sitemaps
- [ ] In GSC, go to **Sitemaps** (left menu)
- [ ] Add: `https://officeaddindevelopment.com/sitemap.xml`
- [ ] Wait for "Success" status (can take 24 hours)
- [ ] Check for coverage errors

**Expected sitemap:**
```
- 92 static pages (already generated by Next.js)
- Blog posts (15+)
- Portfolio projects (50+)
- Service pages (8)
```

### 1.3 Check Indexing Status
- [ ] Go to **Coverage** (left menu)
- [ ] View total indexed pages (should show 90+)
- [ ] Look for errors/warnings in red
- [ ] Click on "Excluded" to see why pages aren't indexed

**Common Issues & Fixes:**
| Issue | Fix |
|-------|-----|
| `Not indexed` | Manually request indexing (click "Inspect" on page) |
| `Discovered - not indexed` | Check robots.txt, meta noindex tags |
| `Blocked by robots.txt` | Verify `/public/robots.txt` allows crawling |
| `Duplicate` | Check canonical tags are correct |

### 1.4 Verify robots.txt
- [ ] View: `https://officeaddindevelopment.com/robots.txt`
- [ ] Should look like:
```
User-agent: *
Allow: /
Sitemap: https://officeaddindevelopment.com/sitemap.xml
```
- [ ] If incorrect, update `src/app/robots.ts` and redeploy

## 📧 PHASE 6: Google Analytics & Monitoring

### 6.1 Set Up Google Analytics 4
- [ ] Go to [Google Analytics](https://analytics.google.com)
- [ ] Create new property for your domain
- [ ] Get Measurement ID (format: `G-XXXXXXXXXX`)
- [ ] Add to your site code OR use Google Tag Manager

**Option A: Direct in Next.js**
```typescript
// Create src/components/analytics/GoogleAnalytics.tsx
import Script from 'next/script';

export function GoogleAnalytics() {
  return (
    <>
      <Script
        src="https://www.googletagmanager.com/gtag/js?id=G-YOUR_ID"
        strategy="afterInteractive"
      />
      <Script
        id="google-analytics"
        strategy="afterInteractive"
        dangerouslySetInnerHTML={{
          __html: `
            window.dataLayer = window.dataLayer || [];
            function gtag(){dataLayer.push(arguments);}
            gtag('js', new Date());
            gtag('config', 'G-YOUR_ID');
          `,
        }}
      />
    </>
  );
}
```

### 6.2 Link GSC to Google Analytics
- [ ] In Google Analytics: Admin → Google Search Console Link
- [ ] Link your property
- [ ] Wait 24-48 hours for data flow

### 6.3 Set Up Alerts
- [ ] GSC → Settings → Search Console Alerts
- [ ] Enable alerts for:
  - Coverage issues
  - Indexing issues
  - Mobile usability problems
  - Security issues

## 🆘 Troubleshooting

### Site Not Indexing?
```
1. Check robots.txt allows crawling
2. Verify sitemap submitted
3. Request indexing manually in GSC
4. Check for noindex meta tags
5. Ensure HTTPS is working
6. Wait 7-14 days (normal)
```

### Poor Performance?
```
1. Check Core Web Vitals in GSC
2. Run PageSpeed Insights
3. Check image optimization
4. Minimize JS bundle
5. Enable compression
6. Check CDN (Vercel default ✅)
```

### Low Traffic?
```
1. Verify keywords are realistic
2. Check keyword search volume
3. Improve content for target keywords
4. Build more backlinks
5. Add FAQ schema
6. Create more blog content
```
